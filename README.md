## 🛠️ Instalación y Configuración Local

### Prerrequisitos
- Node.js (versión 16 o superior)
- npm o yarn

### Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/nelsonalejandro/portafolio.git
   cd portafolio
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Abre tu navegador en `http://localhost:5173`

### Construcción para Producción
```bash
npm run build
```

Los archivos de producción estarán en la carpeta `dist`.

### Variables de Entorno

El proyecto utiliza variables de entorno para configurar servicios externos sin exponer credenciales en el código.

1. Copia el archivo de ejemplo:
   ```bash
   cp .env.example .env
   ```
2. Edita `.env` y reemplaza los valores con tus credenciales reales:
   ```env
   VITE_GROQ_API_KEY=tu_api_key_de_groq
   ```
3. `.env` está en `.gitignore` y **nunca se sube al repositorio**. Solo se versiona `.env.example` con valores de ejemplo.

Puedes obtener una API key gratuita de Groq registrándote en [https://console.groq.com](https://console.groq.com).
